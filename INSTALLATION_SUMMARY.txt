KEY FEATURES:
-------------

Window Management:
  • Dynamic tiling (dwindle layout)
  • Window grouping/tabbing (SUPER+G)
  • Scratchpad workspace (SUPER+S)
  • Split-monitor-workspaces plugin (5 workspaces per monitor)
  • 10 workspaces total
  • Floating & fullscreen modes

Webapps:
  • Chromium-based webapp system
  • Desktop integration with icons
  • Pre-installed: GitHub, ChatGPT, YouTube
  • Easy webapp creation

Snapshots:
  • BTRFS system snapshots using Snapper
  • Automatic snapshots on package updates
  • Easy restore from any snapshot
  • Snapshot management commands

Themes:
  • 14 beautiful themes included
  • Instant theme switching
  • Wallpaper cycling
  • Custom theme support

Tools:
  • Walker app launcher
  • Waybar status bar
  • Mako notifications
  • Screenshot with editing (satty)
  • Screen recording (wf-recorder)
  • Clipboard manager (cliphist)
  • Audio device switcher
  • TUI management apps:
    - bluetui (Bluetooth)
    - impala (WiFi/Network)
    - wiremix (Audio mixer)

INSTALLATION:
-------------

1. cd installation_hyprland
2. ./install.sh
3. Logout and select "Hyprland" from display manager
4. Enjoy!

REQUIREMENTS:
-------------

  • Arch Linux (or Arch-based distro)
  • BTRFS filesystem (recommended for snapshots)
  • Internet connection

DIFFERENCES FROM OMARCHY:
-------------------------

  ✓ No Omarchy-specific binaries
  ✓ No Omarchy fork of Chromium
  ✓ Standard Arch packages only
  ✓ Simplified menu system
  ✓ Snapper-based snapshots
  ✓ All core features preserved
